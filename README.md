# GMaps Crawler

### Requirements

* Python 3.8 32-bit
* Google Chrome
* Chromedriver.exe

### Installation

1. Clone repository (Windows 10):

```
git clone https://github.com/smurilogs/gmaps-crawler.git
```
2. Install the dependencies:

```
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```
